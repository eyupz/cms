\# Kullanılan Teknolojiler



Bu proje geliştirilirken modern web teknolojileri kullanılmıştır.



\## Hugo



Statik web sitesi üretmek için kullanılmıştır.



\### Görevleri



\- Markdown dosyalarını HTML'e dönüştürmek

\- Tema sistemini yönetmek

\- Site yapısını oluşturmak



\---



\## Git



Versiyon kontrol sistemi olarak kullanılmıştır.



\### Sağladığı Avantajlar



\- Değişiklik takibi

\- Commit sistemi

\- Güvenli proje yönetimi

\- Takım çalışması desteği



\---



\## GitHub



Projeyi bulut ortamında saklamak için kullanılmıştır.



\### Özellikleri



\- Uzak repo yönetimi

\- Kod paylaşımı

\- Açık kaynak desteği

\- Deployment entegrasyonu



\---



\## Vercel



Web sitesini internette yayınlamak için kullanılmıştır.



\### Sağladığı Özellikler



\- Otomatik deployment

\- Hızlı hosting

\- HTTPS desteği

\- GitHub entegrasyonu



\---



\## Markdown



İçerik yönetimi için kullanılmıştır.



\### Avantajları



\- Basit sözdizimi

\- Hızlı içerik üretimi

\- Okunabilir yapı

\- Kolay düzenleme

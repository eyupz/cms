# Galeri

Bu bölüm proje geliştirme sürecini temsil etmektedir.

## Geliştirme Aşamaları

### 1. Hugo Kurulumu

Hugo kurulumu gerçekleştirilmiş ve proje klasörü oluşturulmuştur.

-–

### 2. Tema Kurulumu

Ananke teması projeye entegre edilmiştir.

-–

### 3. Sayfa Tasarımı

Çok sayfalı yapı hazırlanmıştır.

-–

### 4. GitHub Entegrasyonu

Proje GitHub reposuna yüklenmiştir.

-–

Site canlı yayına alınmıştır.

# Hakkımda

Merhaba, ben Eyup Zarifoğlu.

Bu web sitesi, statik web sitesi geliştirme sürecini öğrenmek ve uygulamalı olarak göstermek amacıyla hazırlanmıştır.

## İlgi Alanlarım

- Web geliştirme

- Yazılım teknolojileri

- Git ve GitHub kullanımı

- Statik site üreticileri

- Bulut tabanlı yayınlama sistemleri

- Modern frontend araçları

## Bu Projede Ne Yaptım?

Bu projede sıfırdan bir Hugo sitesi oluşturdum. Siteye tema ekledim, farklı sayfalar oluşturdum, menü yapısı kurdum, GitHub reposuna yükledim ve Vercel üzerinden yayına aldım.

## Kazanımlarım

Bu çalışma sayesinde:

- Hugo proje yapısını öğrendim.

- Markdown dosyalarıyla içerik oluşturmayı öğrendim.

- Git komutlarını kullandım.

- GitHub reposu oluşturup proje yükledim.

- Vercel ile canlı yayınlama sürecini deneyimledim.

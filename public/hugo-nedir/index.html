# Hugo Nedir?

Hugo, Go programlama dili ile geliştirilmiş modern ve yüksek performanslı bir statik site üreticisidir.

Markdown formatındaki içerikleri HTML dosyalarına dönüştürerek hızlı ve güvenli web siteleri oluşturur.

## Hugo’nun Özellikleri

- Çok hızlı build süresi

- Tema desteği

- Markdown tabanlı içerik sistemi

- SEO dostu yapı

- Blog ve dokümantasyon desteği

- Kolay deployment

- GitHub ve Vercel uyumluluğu

## Statik Site Avantajları

Statik web siteleri:

- Daha hızlıdır

- Daha güvenlidir

- Sunucu maliyeti düşüktür

- Veritabanı gerektirmez

- Bakımı kolaydır

## Hugo ile Yapılabilecek Projeler

- Ürün tanıtım siteleri

- Teknik rapor siteleri
